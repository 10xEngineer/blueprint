NAME

blueprint - manage blueprints of servers

SYNOPSIS

blueprint [command] [options][...]

DESCRIPTION

blueprint programs manage blueprints of your sandboxes. Blueprints track packages and system files so servers can be rebuilt at any time. Blueprints can be used to generate Puppet modules, Chef cookbooks, or POSIX shell code.

blueprint-list(1)
List all blueprints you can access.
blueprint-create(1)
Turn blueprints into code or save them for later.
blueprint-show(1)
Turn saved blueprints into code.
blueprint-apply(1)
Run the code generated by a blueprint.
blueprint-destroy(1)
Destroy a blueprint.
blueprint-backend(1)
Generate a blueprint for a sandbox.
blueprint-essential(1)
Cache essential package list.
blueprint-mark(1)
Mark arbitrary files for inclusion in a blueprint.
blueprint(5)
Blueprint JSON format.
